# PreventX (PreDiabetes) Mobile Application

This is a React Native mobile application built with Expo for diabetes risk assessment and tracking.

## 🚀 Yerelde nasıl çalıştıracağım? Gerekli kurulum ayrıntıları

Uygulamayı bilgisayarınızda yerel ortamda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

### Gereksinimler (Prerequisites)
1. **Node.js** (v18.x veya daha yeni bir sürüm)
2. **npm** veya **yarn**
3. **Expo Go** uygulaması (Fiziksel cihazınızda test etmek için iOS App Store veya Google Play'den indirebilirsiniz)

### Kurulum Adımları
1. Projeyi bilgisayarınıza indirin (clone):
   ```bash
   git clone https://github.com/Aboubacarelhacen/PreDiabetes_app.git
   cd PreDiabetes_app
   ```

2. Gerekli kütüphaneleri ve bağımlılıkları yükleyin:
   ```bash
   npm install
   # veya
   yarn install
   ```

3. Uygulamayı başlatın:
   ```bash
   npx expo start
   ```

4. Ekranda veya terminalde beliren **QR Kodunu** telefonunuzdaki kameranızla (iOS) veya Expo Go uygulamasından (Android) okutarak uygulamayı anında canlı olarak test edebilirsiniz. Alternatif olarak, bilgisayarınızda iOS Simulator için `i` tuşuna, Android Emulator için `a` tuşuna basabilirsiniz.
